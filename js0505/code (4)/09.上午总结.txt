一、js是什么？
	定义：用来实现前端的页面动态效果和数据交互的编程语言
	官方定义：是一种基于对象和事件驱动的客户端（浏览器段）脚本语言
		
		脚本语言：这个语言需要一个生存环境（浏览器中解释执行的）
		
	js的设计开发的目的：用于表单的数据验证
	
	js的目前的用途：
		1 pc端的页面效果（pc网站前端）
		2 移动端（web页面）开发（H5+css3） ===》app(借助一些框架) 
		3 ajax 数据交互
		4 nodejs（用js的语法 去写服务器端）
		5 js框架（vue react angular） 和插件开发 12306抢票插件
		
	游戏编程	 https://codecombat.163.com/

二、js的诞生1995 开发者 布兰登艾奇  网景和sun公司联合 开发发行 前名livescript 后期更名javascript 

三、js和html 和css

	html结构 
	css样式
	js 行为
	开发的时候强调三者分离
	
	
	js的组成：
		ECMAScript（核心）: 规定的js的语法  var  for if  function...
		es5 ===>es6
		
		DOM: 规定了js如何操作和控制页面中的元素和样式等
		
		BOM: 规定了js如何操作浏览器上的一些行为
		

四、js的标签的书写位置：
			1、外部script标签引入 src=“外部文件.js” （多文件后期频繁使用）
			2 内部script标签中写代码  （现阶段学习使用）
			3 html标签中嵌入js代码 （不推荐）
			
五、三条输出语句
	1 alert() 弹出框
	2 document.write() 往页面中输入内容 支持标签
	3 console.log() 往控制台输出  
	
六、注释：
	单行注释 //
	多行注释： /*  */
	作用：1 对一些复杂逻辑 作解释和说明  2 做功能代码块的分割

八、变量： 在内存中用来存储数据的标识符

	变量的命名规则：
		1 由 字母 数字 下划线 和 $ 组成 不能以数字开头
		2 不能以关键字和保留字命名
		3 区分大小写
	
	变量的命名规范:
		1 命名要有意义 
		2 遵循驼峰式命名法
